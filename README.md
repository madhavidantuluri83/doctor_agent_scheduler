# Doctor AI Scheduler

AI-powered doctor appointment scheduling assistant using Python, Google Calendar API, and LangGraph.

## Features

- Natural language appointment booking
- Availability checking
- Google Calendar integration
- Conflict detection
- Intent routing
- Multi-agent ready architecture

## Example Queries

- Is doctor available tomorrow at 5 PM?
- Book appointment Friday 11 AM
- Schedule consultation next Tuesday 4 PM

## Tech Stack

- Python
- LangGraph
- LangChain
- Google Calendar API
- dateparser
